# Outreach Script Generation

## When to Use
When a rep needs a call script, email draft, or SMS message for a specific account.

## Script Types

### Cold Call Script
Structure:
1. **Opener** — Reference something specific (ranking, review count, recent news)
2. **Value Prop** — Why Power100 matters to THIS company specifically
3. **Bridge** — Connect their current situation to what we offer
4. **Ask** — Clear next step (meeting, demo, info send)

Example opener pattern:
"Hi [CEO Name], this is [Rep Name] with Power100. I've been reviewing the top-ranked [pillar] companies in [metro area], and [Company Name] stood out — your [rating] rating across [review count] reviews is impressive. I wanted to reach out because..."

### Follow-Up Email
Structure:
1. **Subject** — Reference something specific, NOT generic
2. **Opening** — Tie back to previous conversation or trigger event
3. **Body** — One key value point, not a laundry list
4. **CTA** — Single, clear call-to-action
5. **Signature** — Professional, brief

### Re-Engagement Message
When last contact was 30+ days ago:
- Acknowledge the gap naturally, don't apologize
- Lead with NEW value (new intel, new ranking, new opportunity)
- Keep it short — earn the right to longer conversations

### SMS Message
- Under 160 characters when possible
- Include company or CEO name
- Clear CTA
- Professional but not formal

## Key Principles
- **Never sound templated** — Reference specific data points
- **One ask per outreach** — Don't overload
- **Match channel to message** — Complex topics = email. Time-sensitive = SMS. Relationship = call.
- **Use intel** — If we have recent news about them, use it as the opener
- **Don't oversell** — Focus on their needs, not our features
